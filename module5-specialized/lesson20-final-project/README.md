# Lesson 20: Final Project

## Overview

Apply everything you've learned to build a comprehensive AI application.

## Project Options

### 1. AI Research Assistant
Multi-agent system with web search, document analysis, and report generation.

**Features:**
- Web search and scraping
- Document ingestion (PDF, web)
- Multi-agent collaboration
- Report generation
- Source citations

### 2. Code Review Bot
Analyzes PRs, suggests improvements, checks for bugs, generates tests.

**Features:**
- Code analysis
- Style checking
- Bug detection
- Test generation
- Documentation

### 3. Customer Support Agent
RAG over docs, ticket classification, response generation, escalation.

**Features:**
- Document Q&A
- Ticket routing
- Response templates
- Sentiment analysis
- Escalation logic

### 4. Data Analysis Assistant
Natural language to SQL, chart generation, insight extraction.

**Features:**
- NL to SQL
- Query execution
- Visualization
- Insight generation
- Report export

### 5. Custom Project
Design your own project (requires approval).

## Requirements

**Technical:**
- Use LangGraph for orchestration
- Implement at least 3 custom tools
- Include RAG or structured output
- Deploy as API (FastAPI)
- Write unit tests
- Create documentation

**Documentation:**
- README with setup instructions
- Architecture diagram
- API documentation
- Usage examples

**Deliverables:**
1. Working code
2. README.md
3. Architecture diagram
4. Demo video or screenshots
5. Reflection document (what you learned)

## Evaluation Rubric

**Functionality (40%)**
- Works as specified
- Handles edge cases
- Error handling

**Code Quality (20%)**
- Clean, readable code
- Proper structure
- Type hints
- Documentation

**Architecture (20%)**
- Good design decisions
- Scalable
- Maintainable

**Documentation (10%)**
- Clear README
- Good examples
- Architecture explained

**Innovation (10%)**
- Creative solutions
- Advanced techniques
- Polish and UX
