# Lesson 15: MCP Integration

## Theory

Integrating MCP servers with LangChain and LangGraph for powerful AI workflows.

### Integration Patterns
- LangChain agents with MCP tools
- LangGraph workflows with MCP
- Multiple MCP servers
- Dynamic tool discovery

### Benefits
- Modular architecture
- Reusable components
- Easy testing
- Scalable systems

## Challenge

Build an **Agent** that uses multiple MCP servers (filesystem, database, API) to build a data pipeline.

See `demo.py`, `challenge.py`, `solution.py`, and `servers/` for examples.
